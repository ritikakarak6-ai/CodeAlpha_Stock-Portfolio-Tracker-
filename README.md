# Task 2: Stock Portfolio Tracker

## Description
A simple Python command-line application that tracks stock purchases, calculates total investment value based on predefined stock prices, and saves the summary into a text file.

## Features
- Hardcoded dictionary for stock data (AAPL, TSLA, MSFT).
- User input for stock selection and quantity.
- Automated total cost calculation.
- File handling with robust `try-except` blocks for PermissionError.

## How to Run
1. Run `python portfolio.py`
2. Enter the stock ticker (e.g., AAPL) and quantity.
3. Choose 'yes' to save the output in `portfolio.txt`.
